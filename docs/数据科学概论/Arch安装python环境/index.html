在Arch Linux(或Manjaro)中安装python运行环境

方式1#

使用Arch自带包管理器pacman

安装python3

sudo pacman -S python39

安装pip

sudo pacman -S python-pip

方式2#

使用go脚本yay在AUR仓库中安装

注意:不要给yay sudo权限,这是被禁止的

安装python3

yay -S python3

安装pip

yay -S python-pip
